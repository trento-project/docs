=== Using the {tr_mcp}

The {tr_mcp} enables AI-assisted operations by implementing the link:https://modelcontextprotocol.io/introduction[Model Context Protocol] (MCP). This allows MCP-compatible assistants to connect to {trento} and perform monitoring and troubleshooting tasks using natural language.
